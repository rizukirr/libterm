# libterm

Cross-platform terminal UI library — termbox2-compatible API on POSIX and native Windows (Win32 Console API).

Derived from [termbox2](https://github.com/termbox/termbox2). All identifiers renamed from `tb_`/`TB_` to `lt_`/`LT_`.

## Build

```sh
cmake -B build
cmake --build build
ctest --test-dir build
```
